    Proyectos:


    1. GBM Medios (Enero 2024 - Junio 2024)

    GBM Medios es un proyecto desarrollado para la logística del inventariado de medios de cintas del Data Center de GBM Honduras. Este ofrece:

    • Gestión y creación de usuarios (Roles: Root, Admin, Operator)
    • Inventario (Filtrado de datos, agregar, modificar y eliminar inventariado)
    • Generación de reportes (Inventario, Ingreso y retiro de medios en inventario)
    • Accesos según el Rol de cada usuario
    • Historial de acciones (Login, modificaciones del inventario realizadas por el usuario, ingreso y salida de medios)
    • Servicio SMTP (Usado para solicitudes al momento de modificar o eliminar datos del inventario, medio de envío de códigos de verificación)
    • Generación de Token (Tokens aleatorios temporales para login y modificación de inventario que se envían vía correo SMTP)
    ❮ ❯

    Estas son las implementaciones generales, las tecnologías utilizadas fueron:

    • HTML, PHP, CSS y JavaScript en general
    • CSS: Recursos de Bootstrap-icon, Lordicon
    • Base de Datos: SQL Server y como gestor PHPMyAdmin dentro del mismo servidor
    • Framework: NodeJS, Express.js
    • Servidor: Ubuntu Server con Nginx
    • SMTP: Postfix

    3. This Music (Julio 2024 - Julio 2024)

    This Music es una aplicación web que permite buscar música en Spotify y videos en YouTube de manera sencilla. Presenta los resultados en tablas separadas y ofrece la opción de reproducir canciones directamente en la página. Con un diseño moderno y responsivo.

    • Un formulario de búsqueda.
    • Contenedores para resultados de Spotify y YouTube.
    • Botones para cargar más resultados de cada servicio.
    • Estilos básicos para el cuerpo y elementos de navegación.
    • Diseño responsive para diferentes tamaños de pantalla.
    • Animaciones y efectos para la entrada de búsqueda.
    • Estilos específicos para resultados de búsqueda y tablas.
    • Obtención de token de acceso de Spotify y búsqueda de música.
    • Búsqueda de videos en YouTube.
    • Actualización dinámica de los resultados en las tablas.
    • Gestión de botones para cargar más resultados.
    • Reproducción de canciones de Spotify directamente en la página.
