GBM Medios

GBM Medios, es una proyecto desarrollado para la logistica del inventariado de medios de cintas del Data Center de GBM Honduras esta ofrece:

  • Gestion y creacion de usuarios (Roles: Root, Admin, Operator)
  • Inventario (Filtrado de datos, agregar, modificar y eliminar inventariado)
  • Generacion de reportes (Inventario, Ingreso y retiro de medios en inventario)
  • Accesos segun el Rol de cada usuario
  • Historial de acciones (Login, modificaciones del inventario realizadas por el usuario, ingreso y salida de medios)
  • Servicio SMTP (Usado para Request al momento de modificar o eliminar data del inventario, medio de envio de codigos de verificacion)
  • Generacion de Token (Token aleatorios temporales para login y modificacion de inventario que se envia via correo SMTP)
